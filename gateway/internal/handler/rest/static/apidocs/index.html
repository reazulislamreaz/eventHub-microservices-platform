Authentication

Obtain a JWT via login or register, then pass it on protected routes.

Login

POST /api/v1/auth/login
Content-Type: application/json

{
  "email": "admin@eventhub.io",
  "password": "AdminPass123!"
}

Register

POST /api/v1/auth/register
Content-Type: application/json

{
  "email": "alice@example.com",
  "name": "Alice",
  "password": "SecurePass123"
}

REST API

MethodEndpointAuthDescription
POST/api/v1/auth/register—Register user
POST/api/v1/auth/login—Login, get JWT
GET/api/v1/users—List users
GET/api/v1/users/{id}—Get user
GET/api/v1/events—List events
POST/api/v1/eventsAdminCreate event
POST/api/v1/ticketsUserBook ticket
GET/api/v1/users/{id}/ticketsUserUser tickets
GET/health—Health check
GET/ready—Readiness

GraphQL API

Endpoint: POST /query — use the Playground for interactive queries.

OperationTypeAuthDescription
registerMutation—Create account
loginMutation—Authenticate
getUsersQuery—List users
getUserQuery—Get user by ID
getEventsQuery—List events
createEventMutationAdminCreate event
bookTicketMutationUserBook ticket
getTicketsByUserQueryUserUser tickets

Example: list events

{
  "query": "{ getEvents { id title location availableSeats } }"
}

Error format

{
  "error": "Bad Request",
  "code": 400,
  "message": "detailed error message"
}
